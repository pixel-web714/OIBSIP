# OIBSIP

Objectives:

● To understand the use of HTML, CSS, and JavaScript in real-time applications.
● To perform accurate temperature conversions between Celsius, Fahrenheit, and Kelvin.

Steps Performed:

1.Created the basic structure using HTML.
2.Designed the layout and styling using CSS.
3.Added input fields and dropdown for selecting temperature units.
4.Implemented conversion logic using JavaScript.
5.Applied formulas for different temperature conversions.
6.Displayed the result dynamically on user input.
7.Tested the application for different values and responsiveness.

Tools:

HTML – Structure of the web page
CSS – Styling and layout
JavaScript – Logic for temperature conversion
